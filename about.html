Tentang proyek Space Explorer Pro

Tentang Proyek

Space Explorer Pro adalah aplikasi web interaktif yang memungkinkan pengguna menjelajahi keajaiban luar angkasa menggunakan data dari NASA API. Aplikasi ini dibangun dengan teknologi web modern namun tetap sederhana tanpa build tools.

Fitur utama meliputi Astronomy Picture of the Day (APOD), eksplorasi planet dalam tata surya, dashboard analitik dengan visualisasi data, dan galeri gambar luar angkasa.

NASA API

Aplikasi ini menggunakan NASA Open API, khususnya endpoint APOD (Astronomy Picture of the Day) yang menyediakan gambar atau video astronomi setiap hari beserta penjelasannya.

API Key yang digunakan: DEMO_KEY

Untuk penggunaan produksi, silakan daftar API key gratis di api.nasa.gov
